# Mechanical Thesis

0002 — *At Sea* is a recursion stability probe.

Eight cooperative agents assemble a coherent rule-card under clarity constraint using strictly bounded token categories.

The purpose of this prototype is to test whether rule mutation can occur inside a live play loop without collapsing structural clarity or tempo.

This is the Recursion Corner of the Ignition Triangle.

---

# Non-Negotiables

The following constraints define the integrity of this probe and must not be altered during Phase 1:

1. Exactly four token categories:
   - STRUCTURE  
   - ACTION  
   - TARGET  
   - CONDITION  

2. Exactly four tokens per completed rule.

3. Exactly one token from each category per valid rule.

4. No duplicate categories permitted in a single assembly.

5. Passive clarity decay over time.

6. Small clarity cost per input.

7. Heavy clarity penalty for invalid rule construction.

8. Moderate clarity restoration for valid rule construction.

9. Win Condition:
   - Successfully complete 3 valid rules.

10. Loss Condition:
    - Clarity reaches 0.

11. No additional mechanics may be introduced.

These guardrails preserve the integrity of the recursion test.
